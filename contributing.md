Contributing to Humanmark is easy. We welcome contributions from anyone, regardless of experience level.

## How to Contribute

1. Fork the repository and clone it to your local machine.
2. Create a new branch for your changes.
3. Make your changes and commit them with clear and descriptive messages.
4. Push your changes to your forked repository.
5. Open a pull request to the main repository, describing your changes and why they should be merged.

## Types of Contributions

There are many ways to contribute to Humanmark.

- Add new benchmarks
- Add new models
- Localization and translation (This is a complicated one. Most benchmarks are written in English, and changing a benchmark language also affects model scores and the model's ability to understand the benchmark. If you want to translate a benchmark, please contact us first.)
- Bug fixes and bug reports (Obviously, no one likes bugs - bugs are gross. If you find a bug, please report it. If you can fix it, please do, but even if you can't, reporting it is still a huge help.)
